# cart-using-vanillaJS
a e-commerce add to cart functionality using vanilla js
## FEATURE
- [x] Toggle cart visibility
- [x] Add to cart
- [x] Increase/Drecrease quantity
- [x] Remove from cart
- [x] Calculate the total
- [x] Update the cart badge
   

## OVERVIEW
- A eventListener and a function for add items
- A eventListener and a function for remove items
- A eventListener and a function for change quantity
- A function to calculate the total cost & update the badge
